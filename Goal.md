
#
The SMART Components

Specific: Goals should be narrow, clear, and unambiguous, defining exactly what needs to be accomplished.

Measurable: Define criteria, such as metrics or KPIs, to track progress and know when the goal is achieved.

Achievable: Ensure the goal is realistic and attainable given available resources, skills, and time.

Relevant (or Realistic): The goal should align with broader, long-term objectives and be worth pursuing.

Time-bound: Set a clear deadline or time frame to create urgency and focus.

# Customer Baseline

1. Office size: ~1,500–5,000 sq ft (small office / light commercial)
2. People: 10–30 staff
3. Devices per person: 2–3 (laptop + phone + maybe tablet)
4. Shared devices: 5–15 (printers, TVs, VoIP phones, cameras, IoT, conference room gear)
5. Total endpoints: 30–100
6. Internet bandwidth: 300–1000 Mbps down, 20–200 Mbps up (varies; small offices increasingly have fiber/cable)
7. Common requirements: guest Wi-Fi, segmented IoT, reliable VPN, simple admin, basic monitoring/logging, backups, and a path to add services later
