# Zones & DMZ

- Establish a segmented high security **zone for high value assets** and/or OT systems components. (Implied less important zone)

- Protect access to devices within this zone by using specific **firewall
access controls**.

- Establish a **DMZ** for work that must be within the high security zone.

- Limit data traffic to the *Trusted zones* with remote access control.
    - Specific connection controls
    - Specific user controls
